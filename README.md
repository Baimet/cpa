# Champions Pathway Academy 🌟

Champions Pathway Academy is a website built to showcase our mission, programs, gallery, and contact information.  
It empowers athletes through world-class training, mentorship, and community engagement.

## 🚀 Features

- **Home Page** – Introduction to Champions Pathway Academy with an inspiring layout.
- **Programs Section** – Highlights the academy’s sports and training programs.
- **Gallery** – A collection of images from our journey, training, and events.
- **Contact Us Page** – Allows visitors to reach out directly via a Netlify-powered contact form.
- **Responsive Design** – Optimized for desktop, tablet, and mobile viewing.
- **Social Links** – Quick access to our social media platforms.

## 🛠️ Tech Stack

- **Frontend:** React.js
- **Styling:** CSS3
- **Icons:** [React Icons](https://react-icons.github.io/react-icons/)
- **Hosting & Forms:** Netlify

## ⚙️ Installation & Setup

1. **Clone the repository:**

   ```bash
   git clone https://github.com/your-username/champions-pathway-academy.git
   cd champions-pathway-academy

   ```

2. **Install Dependencies:**
   npm install

3. **Start Development Server:**
   npm start

4. **Build for production:**
   npm run build

## Deployment

The project is deployed on Netlify.
To deploy manually:

Run npm run build.

Drag and drop the build/ folder into Netlify, or push the repo and connect via Netlify Git.

Netlify also powers the Contact Form on the site.
